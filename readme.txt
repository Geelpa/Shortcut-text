## Como instalar (modo desenvolvedor) ##

Abra o Google Chrome e vá até:

chrome://extensions/


Ative o Modo do desenvolvedor (canto superior direito).

Clique em “Carregar sem compactar”.

Selecione a pasta do projeto.

A extensão aparecerá na barra de extensões — clique no ícone para abrir o popup e configurar seus atalhos.

## Como usar ##

Clique no ícone da extensão.

No popup, clique em “Adicionar Atalho”.

Preencha os campos:

Nome do atalho: apenas uma identificação (ex: “Mensagem de boas-vindas”).

Código: o texto que você digitará no campo (ex: /c1).

Conteúdo: o texto que será colado automaticamente.

Abra qualquer site e digite o código do atalho (ex: /c1) — o texto será expandido instantaneamente.
 
## Solução de problemas ##

Se o atalho não funcionar, tente recarregar a extensão em chrome://extensions.

Alguns sites podem bloquear scripts de conteúdo — use o botão “Atualizar página” após recarregar a extensão.
